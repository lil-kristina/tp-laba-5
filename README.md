# tp-laba-5
Проект по разработке системы управления автомобильным заводом с использованием паттернов проектирования. 
Система включает в себя производство автомобилей разных марок (BMW, Tesla), управление компонентами, 
проверку качества и различные режимы работы. 

Проект реализует структурные (Adapter, Bridge, Proxy) и поведенческие (Chain of Responsibility, 
Iterator, Strategy) паттерны проектирования. Каждый паттерн решает конкретную задачу в контексте 
автомобильного производства, обеспечивая гибкость, расширяемость и поддерживаемость кода.

Цель проекта - демонстрация применения паттернов проектирования в реальной предметной области 
и организация процесса разработки по методологии Scrum.
